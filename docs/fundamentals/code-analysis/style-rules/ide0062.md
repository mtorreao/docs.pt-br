### <a name="example"></a><span data-ttu-id="730d6-127">Exemplo</span><span class="sxs-lookup"><span data-stu-id="730d6-127">Example</span></span>

```csharp

{
    Hello();

    {
        Console.WriteLine("Hello");
    }
}

{
    Hello();
    void Hello()
    {
        Console.WriteLine("Hello");
    }
}
```
