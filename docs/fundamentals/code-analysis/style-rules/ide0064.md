## <a name="example"></a><span data-ttu-id="dbdd6-120">Exemplo</span><span class="sxs-lookup"><span data-stu-id="dbdd6-120">Example</span></span>

```csharp
// Code with violations
struct MyStruct
{
    public readonly int Value;

    public MyStruct(int value)
    {
        Value = value;
    }

    public void Test()
    {
        this = new MyStruct(5);
    }
}

// Fixed code
struct MyStruct
{
    public int Value;

    public MyStruct(int value)
    {
        Value = value;
    }

    public void Test()
    {
        this = new MyStruct(5);
    }
}
```
